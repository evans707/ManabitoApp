## 1. 環境構築
### 使用言語
html, css, JavaScript, python

### Django
1. Python 3.12.3をインストール\
2. 下記のコマンドを実行し, 仮想環境を作成\
   ``python -m venv venv``
3. requirements.txtからpythonライブラリをインストール\
   ``pip install -r requirements.txt``
4. gitを使用してクローンし, djangoプロジェクトが実行できるか確認
